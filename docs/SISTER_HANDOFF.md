# 妹妹交接導覽

## 打開預覽後先看哪幾頁

1. 先看「總覽」，確認今天人工需要處理的急件。
2. 再看「貼文排程」，確認今天要準備哪一種 IG / FB / Threads 素材。
3. 接著看「任務分工」，分清楚哪些是 Agent 做、哪些是你或平皓要確認。
4. 再看「資料源」和「廣告判讀」，確認今天缺哪份蝦皮、Meta Ads、Momo 或 PChome 報表。
5. 最後看「素材庫」與「出貨交接」，確認要補拍什麼、哪些訂單或出貨問題要人工處理。

## 每天要負責什麼

- 每天固定看貼文排程，確認今天要拍圖、剪影片、補文案還是回覆留言。
- 把缺資料的報表或素材標記清楚，整理成今天要追的清單。
- 確認 X23 素材是否完整：商品圖、短影音、認證圖、評價圖、規格圖、活動文案。
- 把已確認的素材交給中台，並註明用途、截止時間與優先順序。
- 每天下班前回看一次待辦，留下還沒完成的原因與下一步。

## 哪些事情要交給平皓／人工核准

- 平台登入、權限授權、後台綁定與密碼管理。
- 價格、折扣、退款、取消訂單、特殊付款條件。
- 改庫存、下架商品、改保固或對外承諾交期。
- 提高廣告預算、開新廣告活動、換 Pixel 或追蹤設定。
- 使用 KOL、人物照片、評價截圖或還沒確認授權的素材。
- 任何你不確定會不會影響收入、品牌形象或客戶承諾的事情。

## 怎麼把素材交給中台

1. 先把素材放到 X23 指定資料夾或雲端位置，檔名要看得出日期、用途與版型。
2. 在交接訊息中附上素材連結，並寫清楚要做 IG Reels、限動、FB 貼文、廣告圖或商品頁。
3. 一次交付時請包含：素材連結、文案草稿、尺寸需求、截止時間、是否已確認授權。
4. 若素材缺件，先標記「缺素材」或「待授權確認」，不要直接排程發布。
5. 送出後回來確認貼文排程狀態已更新，避免同一份素材被重複交付。
