# GitHub Pages 靜態預覽安全說明

此目錄的 GitHub Pages 版本只用於展示 X23 Ops App 的畫面與操作流程，不是正式營運環境。

## 預覽版範圍

- 只能展示頁面導覽、下單/任務/後台等流程概念。
- 不連接真實平台，也不登入任何外部服務。
- 不儲存、不提交、不顯示任何平台帳號、密碼、API Key、Cookie 或 Token。
- 不包含正式 SQLite 資料庫、報表原始檔、客戶資料或營運匯出檔。

## 禁止放入 GitHub Pages 的內容

- `*.sqlite`、`*.sqlite3`、`*.db` 與相關 journal/WAL/SHM 檔案。
- `.env`、憑證、私鑰、Token、API Key、平台帳密。
- 客戶名單、訂單明細、營收報表、廣告報表、平台匯出原始檔。
- 任何可回推真實帳戶、交易、客戶或供應商的資料。

## 發布前安全檢查

1. 確認 `.gitignore` 已排除資料庫、快取、帳密、憑證與報表原始檔。
2. 執行 `find . -maxdepth 3 -type f` 檢查即將發布的檔案清單。
3. 搜尋敏感字串，例如 `password`、`token`、`api_key`、`secret`、`sqlite`。
4. 確認預覽資料為假資料或範例資料，不能包含真實客戶或平台資訊。
5. 確認 GitHub Pages 只部署靜態檔案；若需要後端、登入或資料庫，應改用正式伺服器環境。

## 備註

預覽版的設計目標是讓協作對象看懂流程，不處理真實交易、不代管平台密碼，也不保存可識別的營運資料。
